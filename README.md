A modern Gutenberg block for displaying "Before/After" comparisons. Perfect for portfolios, case studies, and showcasing results.

## Features
- **Grid Layout**: Display multiple comparisons in a clean grid.
- **Interactive Slider**: Smooth handle for manual comparison.
- **Pagination**: Support for multiple items with easy navigation.
- **Fully Responsive**: Works perfectly on all screen sizes.
- **Easy Customization**: Simple interface within the WordPress editor.

## Installation
1. Download the [latest release](https://sailstudio.com/wp-content/uploads/ss-before-after.zip).
2. Go to **Plugins > Add New** in your WordPress dashboard.
3. Click **Upload Plugin** and select the `.zip` file.
4. Activate the plugin.
